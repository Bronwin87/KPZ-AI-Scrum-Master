# KPZ-AI-Scrum-Master
An application that supports the work of a Scrum Master, utilizing artificial intelligence tools.
<br/>
The app is hosted on Azure: [https://aiscrummaster.azurewebsites.net](https://aiscrummaster.azurewebsites.net/)
